# 基于Vue的青玖科技有限公司响应式官网

### 部署项目

* 安装依赖
```
npm install
```
* 启动项目（自定义命令能够同时启动json-server mock数据）
```
npm run dev
```

### 技术栈：

+ vuecli3
+ vue-router
+ element-ui
+ meta-info
+ prerender-spa-plugin 预渲染页面
+ axios
+ less css预处理器
+ css媒体查询、flex布局、传统浮动布局、div+css布局
+ concurrently 用于支持并行运行 运行vue-cli的时候同时启动json-server
+ vue-lazyload 开启首屏图片懒加载
